# Задача 2. Python-скрипт с внешним API + Docker

Скрипт получает данные о курсах валют из публичного API ЦБ РФ, обрабатывает выбранные валюты USD, EUR и CNY, рассчитывает изменение курса относительно предыдущего значения и сохраняет структурированный отчёт в JSON-файл.

## Локальный запуск

```bash
pip install -r requirements.txt
python main.py
```

EUR — Евро: 85.183 ₽ (-1.1069 ₽, -1.2828%)
