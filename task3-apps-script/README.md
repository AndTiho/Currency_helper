# Задача 3. Apps Script + внешний API в Google Таблице

Скрипт подключается к публичному API ЦБ РФ `https://www.cbr-xml-daily.ru/daily_json.js`, получает курсы валют и записывает обработанные данные в Google Таблицу.

## Логика работы

Скрипт получает данные по валютам USD, EUR и CNY, рассчитывает изменение курса относительно предыдущего значения и процент изменения. Результат записывается в таблицу в структурированном виде: код валюты, название, номинал, текущий курс, предыдущий курс, изменение, изменение в процентах и дата данных.

## Триггер

Используется пользовательское меню Google Таблицы. При открытии таблицы функция `onOpen` добавляет меню `Курсы валют`, из которого можно вручную запустить обновление данных.

## Обработка ошибок

Если API недоступен, возвращает некорректный статус или в ответе отсутствует нужный раздел `Valute`, ошибка записывается в ячейку `J2`.
