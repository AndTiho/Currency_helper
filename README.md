# Задача 1 — Веб-утилита на FastAPI

## Описание

Веб-утилита для просмотра и анализа курсов валют.

Приложение получает данные из внешнего API ЦБ РФ, отображает текущие курсы валют и рассчитывает изменение относительно предыдущего значения.

Используемые технологии:

- Python
- FastAPI
- HTML/CSS/JavaScript
- Jinja2

## Основная логика

- Получение данных из внешнего API
- Обработка JSON-ответа
- Расчёт изменения курса валют
- Отображение данных на веб-странице
- Базовая обработка ошибок API

## Локальный запуск

```bash
cd task1-web

python -m venv .venv
.venv\Scripts\activate

pip install -r requirements.txt

uvicorn main:app --reload
```

После запуска приложение доступно по адресу:

```text
http://127.0.0.1:8000
```

# Задача 2 — Python-скрипт + Docker

## Описание

Python-скрипт получает данные о курсах валют из API ЦБ РФ, обрабатывает данные и формирует структурированный отчёт.

Скрипт:

- получает JSON-ответ
- выбирает валюты USD, EUR и CNY
- рассчитывает изменение курса
- рассчитывает процент изменения
- сортирует данные
- сохраняет результат в JSON-файл
- выводит структурированный отчёт в консоль

## Локальный запуск

```bash
cd task2-python-api

pip install -r requirements.txt

python main.py
```

После запуска создаётся файл:

```text
currency_report.json
```

## Запуск через Docker

Сборка контейнера:

```bash
docker build -t currency-report .
```

Запуск контейнера:

```bash
docker run --rm currency-report
```

# Задача 3 — Google Apps Script + Google Таблица

## Описание

Apps Script подключается к внешнему API ЦБ РФ и записывает обработанные данные в Google Таблицу.

Скрипт:

- выполняет HTTP-запрос через UrlFetchApp
- получает JSON-ответ
- записывает данные в таблицу
- рассчитывает изменение курса
- отображает статус выполнения
- обрабатывает ошибки API

## Используемый триггер

Используется пользовательское меню Google Таблицы.

После открытия таблицы появляется меню:

```text
Курсы валют → Обновить курсы
```

При выборе пункта меню запускается обновление данных.
