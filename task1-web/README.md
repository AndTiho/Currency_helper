# Currency Viewer

Лёгкая веб-утилита для просмотра актуальных курсов валют.

## Стек

| Слой | Технология |
|------|-----------|
| Backend | FastAPI + Uvicorn |
| HTTP-клиент | HTTPX (async) |
| Шаблоны | Jinja2 |
| Внешний API | open.er-api.com (бесплатно, без ключа) |
| Frontend | Vanilla JS + CSS |

## Структура проекта

```
currency-viewer/
├── main.py              # FastAPI-приложение, маршруты
├── currency_service.py  # Сервис запросов к внешнему API
├── requirements.txt
├── templates/
│   └── index.html       # HTML-шаблон (Jinja2)
└── static/
    ├── css/style.css    # Стили
    └── js/app.js        # Клиентская логика
```

## Запуск

```bash
# 1. Установить зависимости
pip install -r requirements.txt

# 2. Запустить сервер (из папки currency-viewer)
uvicorn main:app --reload

# 3. Открыть в браузере
# http://localhost:8000
```

## API-эндпоинты

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/` | Главная страница |
| GET | `/api/rates/{base}` | Курсы для базовой валюты |
| GET | `/api/currencies` | Список поддерживаемых валют |
| GET | `/docs` | Swagger UI (автогенерация) |

## Возможности интерфейса

- Выбор базовой валюты из списка
- Поиск / фильтрация по коду и названию
- Сортировка по любому столбцу (клик на заголовок)
- Время последнего обновления данных
- Обратный курс (1 / курс)
- Обработка сетевых ошибок и неверных запросов
